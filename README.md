# Projeto\_CronogramaProjeto desenvolvido em Python para enviar automaticamente por e-mail as tarefas programadas para o dia, utilizando um banco de dados em CSV e GitHub Actions para execução diária.O cerne do script é composto por 4 arquivos. O primeiro é um arquivo é o database.csv, ele é responsável por guardar a lista de tarefas, a primeira coluna guarda o dia do qual a tarefa pertence, a segunda coluna guarda o horário, e a terceira coluna guarda as tarefas que ocorrem em seu respectiva dia e horário(OBS: cada linha carrega uma lista com as tarefas especificas, exceto a primeira, que funciona como uma facilitação para a leitura do arquivo em forma de planilha).O segundo arquivo é o Armazenamento.py, ele é o arquivo menos utilizado, que consiste em adicionar tarefas ao primeiro arquivo. Ele é composto por duas funções principais, que são interdependentes e se unem no final por um bloco condicional que verifica se o usuário quer adicionar informações no databese.csv. A primeiro função executa um loop que pergunta o horário, e a tarefa do horário, assim, adiciona-os a uma lista composta por outras listas. A segunda função é responsável por adicionar as informações da lista composta no arquivo database.csv. Por fim, há um bloco condicional que é executado se o usuário quiser adicionar informações ao primeiro arquivo, se for verdadeiro, o referido executa as duas primeiras funções.O terceiro arquivo é o Leitura.py, que têm a função de ler o arquivo e verificar se o primeiro valor é o dia de hoje(o dia atual), se for o dia atual ele adiciona a tarefa e o horário a uma lista. O quarto arquivo é o Enviar.py, que têm a função de importar a lista do terceiro arquivo e enviar por e-mail.Os demais arquivos do projeto são usados para a execução da tarefa no git actions e para a guardar as senhas usadas no envio do E-mail.
